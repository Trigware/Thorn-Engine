# Thorn Engine
Welcome to the documentation wiki for the **Thorn Engine**, an _(editorless)_ game engine created using **SDL2**, which is being made for development of 2D games in C++.
Planned for development are these features:

* Actor based **Entity-Component System** with UUIDs, archetypes, scenes are more _(in development)_
* Smart asset import system with **enums as identifiers**, instead of strings _(in development)_
* Various input systems such as **actions** and **button presses** _(no progress yet)_
* **Time API** with timers, stopwatches, delta time and more _(no progress yet)_
* Collision detection with simple **2D physics** _(no progress yet)_
* System for saving game data into a parsable format _(no progress yet)_
* An expressive dialog tree system for narrative-heavy games _(no progress yet)_
* And much, much more!

Please checkout the Thorn Engine Wiki for API documentation and instructions!
