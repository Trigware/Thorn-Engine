Welcome to the documentation wiki for the Thorn Engine, an (editorless) game engine created using SDL2, which is being made for development of 2D games in C++.
Planned for development are these features:

* Actor based Entity-Component Systems with UUIDs, archetypes, scenes are more
* Smart asset import system with enums as identifiers, instead of strings
* Various input systems such as actions and button presses
* Time API with timers, stopwatches, delta time and more
* Collision detection with simple 2D physics
* System for saving game data into a parsable format
* An expressive dialog tree system for narrative-heavy games
* And much, much more!

Please checkout the Thorn Engine Wiki for API documentation and instructions!
